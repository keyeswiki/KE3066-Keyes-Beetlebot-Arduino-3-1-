安装前需调节舵机角度为180°

将舵机的线接到PCB上板的S2接口，如下表:

====== =========
舵机   小车PCB板
====== =========
棕线   G
红线   5V
橙黄线 S2（A0）
====== =========

.. code:: c++

   #include <Servo.h>
   Servo myservo;  // create servo object to control a servo

   void setup() {
     myservo.attach(A0);  // attaches the servo on pin A0 to the servo object
   }

   void loop() {
     myservo.write(180);  // tell servo to go to position
   }
